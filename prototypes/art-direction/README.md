# Art Direction Prototype

## How to Test

1. Set main scene to `res://prototypes/art-direction/ArtDirectionPrototype.tscn`
2. Press F5
3. Use **← →** arrow keys to cycle through 4 screens:
   - **Main Menu** – Title header on pink ribbon, play/skins buttons on colored headers
   - **Gameplay HUD** – Notebook-style level header, cat-head move counter, action buttons
   - **Level Complete** – Stars, starburst overlay effect, colored button stack
   - **World Map** – Tab navigation with notebook headers, level grid with star ratings
4. Press **ESC** to quit

## Core Question

Do the new draft UI assets (headers, stars, HUD elements, overlays) create a cohesive
kawaii visual identity when assembled into actual game screens?

## Assets Tested

- `headerBow_*.png` – Ribbon-style headers (pink, purple)
- `header_*.png` – Flat headers (purple, cream, cyan)
- `headerAsim_*.png` – Asymmetric headers (cyan)
- `notebook_*.png` – Notebook-style tab headers (cream, pink)
- `star_gold.png` / `star_grey.png` – Star rating icons
- `move-counter-bg.png` – Cat-head shaped HUD background
- `effect_*.png` – Starburst overlay effects (pink, cyan)
